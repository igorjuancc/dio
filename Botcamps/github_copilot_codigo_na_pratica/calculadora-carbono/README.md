# Calculadora de Emissões de CO₂

Um projeto completo de calculadora de emissões de dióxido de carbono desenvolvido para o Bootcamp de GitHub Copilot.

## Funcionalidades

- Cálculo de emissões de CO₂ entre duas cidades.
- Seleção do meio de transporte: bicicleta, carro, ônibus ou caminhão.
- Preenchimento automático ou manual da distância.
- Comparação das emissões entre diferentes meios de transporte.
- Estimativa de créditos de carbono e valor monetário correspondente.

## Tecnologias

- HTML, CSS e JavaScript puro (ES6+)
- Design responsivo, Mobile First
- CSS Grid e Flexbox

## Como usar

1. Abra o arquivo `index.html` em um navegador moderno (Chrome, Firefox, Safari ou Edge).
2. Preencha a cidade de origem e destino.
3. Escolha o meio de transporte.
4. Clique em **Calcular Emissão** para ver os resultados.
